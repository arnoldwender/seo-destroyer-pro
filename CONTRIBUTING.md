# Contributing to SEO Destroyer Pro

Welcome, fellow SEO saboteur! So you want to help make the internet's search rankings even worse? We appreciate your dedication to the cause.

## Getting Started

```bash
git clone https://github.com/arnoldwender/seo-destroyer-pro.git
cd seo-destroyer-pro
npm install
npm run dev
```

The app should be running at `http://localhost:5173`. Start destroying.

## How to Contribute

1. **Fork the repo** and create a feature branch (`feat/even-worse-seo-advice`)
2. **Make your changes** — new destruction methods, funnier anti-recommendations, wilder certificates
3. **Test locally** — make sure your destruction actually works
4. **Submit a PR** with a clear description of what you broke (on purpose)

## Code Style

- TypeScript everywhere, no `any` unless you're feeling destructive
- React functional components with hooks
- Tailwind CSS for styling
- Keep the humor smart, not mean-spirited — we're roasting SEO, not people
- If your joke needs an explanation, it's not funny enough

## Ideas We Love

- New destruction categories (image SEO crimes, meta tag atrocities, etc.)
- More absurd certificate designs
- Easter eggs that reward curious users
- Accessibility improvements (the irony of making an SEO destroyer accessible is not lost on us)

## Bug Reports

Found a bug? Open an issue. Describe what happened, what you expected, and what actually went wrong. Screenshots help. If the bug accidentally improves someone's SEO, that's a critical severity issue.

## Feature Requests

Got an idea for a new way to destroy SEO? Open an issue with the `enhancement` label. Bonus points if it made you laugh while writing it.

## The Golden Rule

This is a joke app. Keep it fun, keep it clever, and remember: no actual websites were harmed in the making of this tool.
